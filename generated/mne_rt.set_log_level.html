mne_rt.set_log_level#

mne_rt.set_log_level(verbose: bool | str | int | None) → None[source]#

Set the ANT (and MNE) logging level.

Parameters:
verbosebool | str | int | None

Desired verbosity level.

  • None — no change

  • True — "INFO"

  • False — "WARNING"

  • "DEBUG", "INFO", "WARNING", "ERROR"

  • An integer logging level

See also

mne.set_log_level

Set MNE’s own logging level.
